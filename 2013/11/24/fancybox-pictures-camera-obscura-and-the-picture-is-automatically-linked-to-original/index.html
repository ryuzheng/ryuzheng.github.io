fancybox图片暗箱以及图片自动链接到原图

By 郑泽鑫

11月 24 2013 更新日期:5月 22 2014

文章目录

知更鸟的图片暗箱用的是fancybox，而我用的是pirobox，其实两者添加的方法都是大同小异的，大家自己喜欢什么样的风格，可以自己选择。

DEMO：

fancybox：fancybox.net

pirobox：http://www.pirolab.it/pirobox/

图片自动链接到原图的代码来自于知更鸟的“为WordPress文章图片自动添加链接到原图”，相信大家都很羡慕知更鸟的图片暗箱，甚至视频暗箱等各种特效，而知更鸟近日发布的添加链接到原图，可说是更加方便了。其实要为图片添加暗箱特效，都要给图片一个特别的属性，比如“class=fancybox”，这样，而且还要有链接到大图。

但是，我们知道，知更鸟给了我们“自动添加图片属性”的代码（请自己修改一下，代码才有效）

`//自动添加标签属性 

add_filter('the_content', 'pirobox_gall_replace');   

function pirobox_gall_replace ($content)   

{   global $post;   

    $pattern = "/<a(.?)href=('|\")([^>]).(bmp|gif|jpeg|jpg|png)('|\")(.?)>(.?)<\/a>/i";   

    $replacement = '<a$1href=$2$3.$4$5 class="pirobox_gall"$6>$7</a>';   

    $content = preg_replace($pattern, $replacement, $content);   

    return $content;   

} `

所以我们只需要给图片添加链接就行了，想一想，现在有了自动代码，现在只需要插入图片，就自动有了暗箱特效，而以前没有添加原图链接的，现在都不用烦恼了——

`<script type="text/javascript">  

$(function() {  

    $('.entry-content img').each(function(i){  

        if (! this.parentNode.href) {  

            $(this).wrap("<a href='"+this.src+"' class='highslide-image' onclick='return hs.expand(this);'></a>");  

        }  

    });  

});  

</script>  `

将以上代码添加到主题的header.php中，在/head前面就行。

值得一提的是，很多人都不成功，那是因为上面文字中红色的部分，在不同主题中一般都不一样，所以要根据情况来修改，比如我的这个主题就是“.entry-content img”（别忘记那个点号“.”），但其他主题可能就是“.entry img”

我教大家一个方法来看自己的主题到底要怎样修改。打开自己主题的“style.css”文件，然后搜索“.entry”，找到有没有一个是带有“img”在后面的属性，如下图：

看到了吧，所以一样修改就行。但是最后我还是要说一说，其实这个自动添加链接代码虽好，但是由于调用顺序的原因（这个是我自己猜的），所以有时候原图是连接到了，但是图片暗箱却没了。

如果你遇到这种情况，我的解决方法就是，别自己添加js调用pirobox了，直接使用wordpress的插件就行了，wordpress的图片暗箱有很多插件，而使用了插件后，也许是调用顺序的原因，100%成功运行了，但是仅仅支持图片功能而已，而其他功能就没有了。

在此我也希望大家可以给出更好的解决方法。​
