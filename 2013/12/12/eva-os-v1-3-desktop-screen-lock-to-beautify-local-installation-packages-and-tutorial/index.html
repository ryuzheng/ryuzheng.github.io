制作说明：

本主题是基于Zooper的强大代码换算能力制作的，原理和PC的雨滴是几乎一样的，由于Zooper的某些高级代码只能在4.1以上的系统中运行，可能导致在4.0系统中某些功能无法实现，具体测试为运营商检测乱码，系统内存监控失灵，其它情况未知。

Zooper具有一定的分辨率自动适应能力，但是由于本皮肤使用了许多的像素换算函数，导致了这个适应能力变得很弱，最佳适应分辨率为720X1280，其余的分辨率将会在之后进行详细的说明。

本主题的桌面部分内存占用为30M，其多余内存的占用取决于第三方桌面的使用，安装包中的buzz桌面的内存占用70-130M左右，视不同机型有所波动。本人降低了主题的刷新帧率（十秒更新一次基本常数），所以数据更新会有所减缓，但是耗电量大幅下降。

关于第三方桌面的取舍，理论上，任意一个能够全屏，能够改变Widget大小的桌面，都可以使用本主题，楼主这里使用Buzz只是个人习惯，其它人可考虑使用其它的桌面。

看似很麻烦，其实也就10分钟就搞定了，按照顺序来，操作十分简单。

Buzz桌面自带的三个手势，两指缩放预览全部桌面，上拉进入程序列表，下拉通知栏。

教程正文部分：

基本安装（一定要注意顺序！）：

方法一 

从buzz的主页进入我的主页云安装

我的主页http://www.homepackbuzz.com/lionsky 云安装

最新版本是1.22版本，距离1.3版本还是有一些差距的，充电动画有一些小bug。

没有锁屏和开机动画，添加方法见本地安装。

方法二

本地安装

1.APK安装，依次安装根目录的三个apk，分别是buzz桌面（你想装别的桌面也行），Zooper插件代码库，以及锁屏，然后从google play下载一个叫做“media utilities”的插件，这个插件下载安装后，进去设置，勾选Zooper模式即可，这是为了使得Zooper制作的桌面播放器能够正确读取不同的音乐播放器的歌曲信息，基本上主流的播放器都可以正常使用。

以上全部为正版程序，弹谷歌验证的可以考虑下载破解版谷歌市场，具体手段百度即可。

2.素材导入，Zooper代码集成里面的那些ZW文件，就是楼主写好的Zooper皮肤代码，直接复制到“内存设备\ZooperWidget\Templates”下即可，然后是字体，同样复制到“内存设备\ZooperWidget\Fonts”文件夹下即可。锁屏解锁动画的导入，将EVA Lock Pack复制到“内存设备\data\com.teslacoilsw.widgetlocker\themes”文件夹下。

3.开机动画的导入。都一样，没啥可说的，看准分辨率即可。

4.桌面的正式加载。

进入buzz桌面，在第一个页面把行列数设成12X12，关闭状态栏，导航栏和dock，成为全屏状态。添加3个新页面（一共4个）。在第一个页面创建4个zooper widget大小随意，创建后千万不要去点它，只能长按，编辑大小，拉到和屏幕同宽，行宽度由上到下四个widget分别是2行，5行，4行，1行。从第一个widget点进去，选择sdcard，选择EVA 0，加载后返回桌面，由上至下四个插件的导入skin分别是EVA0 EVA1 EVA2 Dock。

之后的2,3,4页，各创建一个zooper widget，拉伸到全屏，导入nerv1 nerv2 nerv3.

出现分辨率不适应的解决手段。

情况1.有虚拟键 修改build.prop删除虚拟键，改用lmt或者gmd之类的侧边快捷键。

情况2.分辨率不是720p。 

我说了，zooper有一点的分辨率适应能力，这个适应能力在不同机型上表现是不同的。点击错误缩放的widget，可见再次进入了设置界面，此时zooper会自动刷新调整一次分辨率，若失败，则只能手动进入layout，修改各个区组的scale（缩放参数），这个非常麻烦，请自行谷歌教程。

桌面的快捷程序设置。

点击widget进入设置界面，选择layout，可见各个区组，以eva 0 为例，其中的bitmap就是那个搜索图样，rect就是右边的那个语音图标，点进去后，可见一个Module Ontap的选项，里面便可以设置点击触发行为，包括打开程序（applist）。

一般情况下

Bitmap就是图片素材，在nerv X的几个皮肤中，bitmap下面显示的（1）之类的序号的，就是图标，右边的小方块勾选后可以fouce这个图标，从而知道这是哪个图标，然后设置方法同上，这些操作和原理与pc版的雨滴是一样的。

硬盘监控默认为es文件管理器，机密默认为桌面设置。

天气城市的设置

拥有天气的widget有eva1，nerv3，还有锁屏的nerv lock

进入widget，修改location，输入城市拼音点确定。

禁用widget的点击触发设置。

点入widget 选择 widget ontap action 选择 disabled即可。刷新周期五秒后生效。

日常设置

由第一页最上面的机密进入，选择configure widgets 可见当前所有的运行中的widget，选择对应的，进去修改即可

三．锁屏的添加

进入widget locker，右上角设置换锁屏壁纸，长按解锁，编辑，换nerv lock皮肤，左相机又解锁（人懒，就做了2个图标）

然后添加一个zooper widget 拉到最大（无法越界到锁屏按钮的界面，大概4X4左右），然后同样导入sdcard中的nerv lock皮肤即可。

天气设置和禁用点击触发设置同上。

其它素材

紫色面板，有兴趣的可以自己替换。

一堆EVA图标，可以用于替换程序列表里面的应用图标。

已知，HTC兼容性最佳，华为测试机不说了，小米兼容性较好，三星较差。LG未测试。
