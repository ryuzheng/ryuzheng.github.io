从《Building Facebook-like Vertical Timeline using Javascript, PHP and MySQL》修改支持中文、汉化部分文字

By 郑泽鑫

12月 14 2013 更新日期:5月 22 2014

文章目录

前些日子很迷恋类似于Timeline的一些插件、主题，所以上网搜了很多，这几天会一一介绍一下，效果都是不错的；其实这个东西我已经修改了很久，一直都没有时间放出来；昨天终于放上github了，其实修改的内容很简单。

(1)最初的版本：http://builtbybalance.com/的"[balanceTimeline](https://github.com/wnyc/Timeline)&quot;

(2)《Building Facebook-like Vertical Timeline using Javascript, PHP and MySQL》的版本 demo (使用方法可参考这里非常详细)

(3)目前已支持中文 现支持中文版 demo 下载地址

修改内容：

(1)首页的四个按钮——展开所有 收起所有 新的在前 旧的在前（汉化）;

(2)"data.php"添加"mysql_query ( "set names utf8" );"以支持中文;

(3)"timeline.sql"中的"ENGINE=InnoDB DEFAULT CHARSET=latin;"修改为"ENGINE=InnoDB DEFAULT CHARSET=utf8;"已支持中文
