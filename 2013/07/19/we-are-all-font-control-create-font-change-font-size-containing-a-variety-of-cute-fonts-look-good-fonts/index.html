安卓

我们都是字体控！自己创作字体、修改字体大小，内附各种萌字体好看字体

今时今日，玩安卓怎么能不玩字体，但是其实换字体是一件挺危险的事情，玩得好，开心舒服；玩的不好，折腾又累，严重者还要重新刷机。

下文将会提供几种换字体的方法。

但无论如何，还是有注意事项的


DroidSans.ttf              默认数字和英文字体(无特殊格式)
DroidSans-Bold.ttf         默认数字和英文粗体(无特殊格式)
DroidSansFallback.ttf      默认中、英、数字体(所有字库均找不到匹配字体后会查找此字体库)
把自己喜欢的字体改名为DroidSansFallback.ttf （替换原有的，可以把原有的随便改个别的名）


1、手机必须要完全ROOT。
2、替换之前必须备份好之前的字体文件，实在不会的就把**systemfonts目录整个备份好

3、下载字体包解压后得到DroidsansFallback、DroidSans.ttf、DroidSans-Bold.ttf 文件（不一定每种字体都有这三个文件，有哪几个就修改哪几个，文件后缀全部为.ttf）。



第一种 最安全方法 手动

最安全的办法莫过于自己手动修改了，什么样的软件都比不上自己的可靠，但是这种方法比较繁琐
1、用RE管理器，长按字体文件选择复制，然后粘贴到手机存储system目录下（注意：RE管理器“载装读/写才能粘贴，RE右上角的按键。）然后长按字体文件选择修改权限，改成下图一样，点确定。

 



2、改好权限之后将字体再剪切到手机存储systemfonts目录下，提示覆盖，点确定。
3、重启手机


第二种 比较方便管理又安全的方法 360优化大师

不是给360打广告，而是优化大师确实比较安全，比较方便

下载地址 360手机优化大师

把字体文件复制到储存卡/360/fonts目录

然后在手机端打开360手机优化大师

选择字体美化



看到许多字体


随便点击一个进入会看到每个字体的预览图


点击应用，会弹出来“确定要更换当前字体？” 点击确定，然后按提示操作





 



360本身还提供了许多字体给各位，地址链接在此

http://shouji.360.cn/pop/360sysopt/fonts.html?1343969414


第三种 最便捷但不安全的，但却很受欢迎 软件流

这里推荐使用“爱字体”，使用方法和360手机优化大师大同小异，就不再多介绍了


下载地址 爱字体

值得一提的是 爱字体的网站是不错的 有很多分享的好字体也可以上去design自己的字体 http://aifont.com/




下面介绍一个软件 可以自己创作字体，修改字体大小

名字很长Font Creator Program - Windows TrueType Font Editor

不过没关系，同样提供下载地址 字体修改工具

转自机锋 作者2219416

提取该文件，用字体修改器打开：


然后点：“格式—设置—布局—单位” 处调整字体大小，（修改数值越小，字体越大，具体要取值多少示自身情况需要而定）


然后保存文件，再用RE把文件放回原处，选择权限

最后重启手机生效~**
