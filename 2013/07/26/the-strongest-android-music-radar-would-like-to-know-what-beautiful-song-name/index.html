安卓最强劲音乐雷达，想知道好听的歌叫什么名字？

By 郑泽鑫

7月 26 2013 更新日期:5月 22 2014

文章目录

走过街头，一时听到上次就听到过的很好听的歌，心里很想找到它的名字，但是掏出手机，用了手机上的音乐雷达却发现结果差强人意啊，心中不免一阵失望。。。。。。

但是，今天我将会介绍给大家两款最强劲的音乐搜索软件，把身边所有好听的歌都一网打尽，放心，我绝对不会推荐以下这几款弱爆软件的。这几款真的不具备生存的价值。

为了不得罪人我还是不点名了


第一款 track ID



索尼（索爱）手机一直坚持拥有的原生软件，可见其实力，所有才拥有这么强大的生命力。

使用简单、反应快、命中高，而且比较美观，绝对是强中之强啊！




这里我就提供谷歌官方市场给大家 （点击图片进入）

有需要的也可以进我的百度网盘下载（点击图片进入）


第二款 Shazam


这款软件虽然偏向于欧美，但是歌曲命中率奇高啊，是我用过中的最强劲的音乐雷达。所以当Track ID找不到时，这款软件就是最好的替代!


但是这款软件最好就是使用已付费汉化版，而且界面没有
Track ID 那么好看。
