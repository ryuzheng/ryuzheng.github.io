用pirobox v1.1集成图片暗箱特效到wordpress

相信大家都看过知更鸟的“将图片暗箱特效集成到WordPress主题中”这篇文章，里面用的就是pirobox这个插件，但是鸟哥的这个pirobox太旧了，现在的pirobox不单单有图片暗箱，甚至还有网页暗箱、视频暗箱功能，所以我才要写这篇文章。其实方法也是很简单的，而且所有图片类的都可以自动添加pirobox的标签，但是其他类别的我还没做到。

首先大家可以上pirobox的官网看看demo

demo地址 http://www.pirolab.it/pirobox/feb_2011.html

下面开始正文部分

首先，下载pirobox v1.1文件，你可以选择我在这里的链接或者官网下载

下载地址（点击下载）

下载后解压，然后将文件夹内的所有文件放到你所使用主题的目录中。

其次，打开主题header模版文件，在</head>前面，添加如下代码：

 

`<link href="<?php bloginfo('template_url'); ?>/css_pirobox/style_5/style.css" rel="stylesheet" type="text/css" />

<script type="text/javascript" src="<?php bloginfo('stylesheet_directory'); ?>/js/jquery_1.5-jquery_ui.min.js"></script>

<script type="text/javascript" src="<?php bloginfo('stylesheet_directory'); ?>/js/pirobox_extended_feb_2011.js"></script>

<script type="text/javascript">

$(document).ready(function() {

    $.piroBox_ext({

    piro_speed :700,

    bg_alpha : 0.5,

    piro_scroll : true,

    piro_drag :false,

    piro_nav_pos: 'bottom'

});

});

</script>`

 

最后，我重新修改了知更鸟的自动添加标签中的正则表达式，以让它符合新版的需要。

将下面代码放在主题的functions.php中：

 

`//pirobox v1.1自动添加标签代码 

add_filter('the_content', 'pirobox_gall_replace');   

function pirobox_gall_replace ($content)   

{   global $post;   

    $pattern = "/<a(.?)href=('|\")([^>]).(bmp|gif|jpeg|jpg|png)('|\")(.?)>(.?)<\/a>/i";   

    $replacement = '<a$1href=$2$3.$4$5 rel="gallery" class="pirobox_gall" title=""$6>$7</a>';   

    $content = preg_replace($pattern, $replacement, $content);   

    return $content;   `

}  

 

当你完成这些步骤后，就可以使用到最新的pirobox的图片暗箱特效了，记住插入的图片要给它们添加链接地址啊！这样图片暗箱才能打开它的原图！ 

下面就是图片的效果啦！非常炫！

（点击打开大图吧）

下面这个是视频暗箱测试！点击打开

大家也许很羡慕知更鸟的视频暗箱特效吧，现在自己也可以做一个啦！不过我用的是pirobox，他用的是fancybox，其实都是原理一样的。

下一次我会讨论一下fancybox以及知更鸟分享的“图片自动链接”的代码。
